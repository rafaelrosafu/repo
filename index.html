Setup instructions

Before using these packages you need to add this repo to your source list, to do that just follow these instructions:

wget -q -O - http://apt.rafaelrosafu.info/rafaelrosafu.key | sudo apt-key add -
echo 'deb http://apt.rafaelrosafu.info/ raring main' | sudo tee /etc/apt/sources.list.d/rafaelrosafu.list
sudo apt-get update
